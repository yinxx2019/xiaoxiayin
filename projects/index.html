Connect Four

Python3, Python Mode for Processing

GitHub Repository

This project is a graphical game of Connect Four in Python/Processing. The object of the game is to let a player drop disks into a column from above and get four in a row horizontally, vertically, or diagonally before the computer opponent (driven by the minimax algorithm) does. If the board fills up and neither player has four in a row, it's a draw. At the end of gameplay, no matter how it ended, the program will prompt the user for a name and save the name and score in a text file. A testing suite is included in the GitHub repository.
